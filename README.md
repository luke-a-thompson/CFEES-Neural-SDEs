# Explicit and Effectively Symmetric Schemes for Neural SDEs
This repository contains code for the paper _Explicit and Effectively Symmetric Schemes for Neural SDEs_.

## Related repositories
Some relevant code exists in other repositories to ease reuse, or because they require substantially different environments.

**Experiments**
* [RoughBench](https://github.com/luke-a-thompson/RoughBench) - Contains the rough volatility experiments used in the paper

**Theory**
* [Kauri](https://github.com/daniil-shmelev/kauri) - Tools used to derive EES Runge-Kutta and commutator-free methods

**Numerics**
* [Georax](https://github.com/luke-a-thompson/georax) - Geometric numerical integration for diffrax including manifold 2N
* [Diffrax-lowstorage](https://github.com/luke-a-thompson/diffrax-lowstorage) - 2N methods for Diffrax
* [EffectivelySymmetric.jl](https://github.com/luke-a-thompson/EffectivelySymmetric.jl) - Effectively symmetric RK schemes in Julia
